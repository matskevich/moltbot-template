**Хочу поделиться своим AI-ассистентом в Telegram — он помнит всё, работает на твоём сервере, не зависит от чужих облаков.**

Что он умеет:
- отвечает в Telegram через Claude (Anthropic)
- помнит весь контекст между сессиями (Gemini embeddings, бесплатно)
- живёт на твоём VPS — полный контроль, никакие данные не утекают
- развивается: создаёт навыки, формирует личность, учится

Setup ~30 минут. Нужен VPS (~€4/мес Hetzner) + API ключи.

**Как начать:**

1. Поставь Claude Code (если ещё нет):
   - **Cursor** — уже встроен (панель снизу)
   - **VS Code** — расширение "Claude Code"
   - **Терминал** — `npm install -g @anthropic-ai/claude-code && claude`

2. Создай папку и открой в Claude Code:
   ```
   mkdir ~/moltbot && cd ~/moltbot
   ```

3. Вставь этот текст в Claude Code — он проведёт через весь setup:

```
════════════════════════════════════════
СКОПИРУЙ ОТСЮДА:
════════════════════════════════════════

Привет! Помоги развернуть личного AI-ассистента.

1. Склонируй шаблон:
git clone https://github.com/matskevich/openclaw-infra.git my-moltbot
cd my-moltbot

2. Прочитай FRIEND-START.md — там сценарий как вести меня по шагам

3. Потом SETUP.md — техника: VPS, ключи, деплой

Режим работы:
- веди пошагово: один шаг → один вопрос
- спрашивай только нужное: VPS, API ключи, Telegram
- ключи НЕ в чат и НЕ в репо — только .env на сервере
- если команда опасная (снос/права/токены) — стоп и предупреждение
- ПЕРВЫМ ДЕЛОМ: помоги настроить spending limits на API ключи

Начинай: спроси про VPS (есть ли, где) и какие ключи готовы.
════════════════════════════════════════
```

---

**Стоимость:**
- VPS: ~€4/мес (Hetzner CX22)
- Claude API: $5-20/мес (зависит от использования, ставь лимит!)
- Gemini (память): бесплатно
- Telegram бот: бесплатно
